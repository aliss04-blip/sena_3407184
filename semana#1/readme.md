# Proyecto Semana 1 - Ficha de Presentación

Dominio: Gestión de Consultas Veterinarias

Este proyecto consiste en un script en JavaScript que muestra
una ficha de una consulta veterinaria en la consola.

Se utilizan conceptos básicos de JavaScript:

- console.log()
- tipos de datos (string, number, boolean)
- comentarios en español
- salida organizada en consola

## Cómo ejecutar

1. Abrir la terminal
2. Ir a la carpeta del proyecto

cd 3-proyecto

3. Ejecutar el archivo

node starter/script.js
