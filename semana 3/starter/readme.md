# Proyecto Semana 03: Calculadora de Dominio

> **🎯 ÚNICO ENTREGABLE**: Este proyecto es el **único entregable obligatorio** para aprobar la semana.

## 🎯 Objetivo

Construir una **calculadora multi-operación** aplicada al dominio **Gestión de Consultas Veterinarias**, usando todos los operadores aprendidos esta semana: aritméticos, de asignación compuesta, de comparación estricta y lógicos.

## 📋 Dominio del Proyecto

**Dominio**: Gestión de Consultas Veterinarias

Este proyecto simula cálculos básicos que pueden realizarse en un sistema de gestión veterinaria, como:

- cálculo del **total de consultas atendidas**
- **ingresos generados por consultas**
- **venta de medicamentos**
- **capacidad de atención diaria**
- aplicación de **descuentos o promociones**

El objetivo es aplicar operadores de JavaScript para representar situaciones reales dentro de una clínica veterinaria.

## ✅ Requisitos Funcionales

La calculadora incluye:

1. **Operaciones aritméticas** (`+`, `-`, `*`, `/`, `%`)
   - cálculo de ingresos por consultas
   - total de ventas de medicamentos
   - promedio de ingresos por consulta
   - consultas restantes del día

2. **Operadores de asignación compuesta** (`+=`, `-=`, `*=`, `/=`)
   - actualización del total de ingresos
   - aplicación de descuentos
   - incremento por servicios adicionales

3. **Comparación estricta** (`===`, `!==`, `>=`, `<=`, `>`, `<`)
   - validación de número exacto de consultas
   - verificación de capacidad máxima del día

4. **Operadores lógicos** (`&&`, `||`, `!`)
   - condiciones para aplicar descuentos
   - verificación de promociones
   - validación de ausencia de consultas

5. **Numeric separators** para valores grandes (`50_000`, `100_000`)

6. **`console.log()`** para mostrar resultados claros en consola
